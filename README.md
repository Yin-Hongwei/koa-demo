## Build Setup

下载本项目到本地

```
git clone https://github.com/Yin-Hongwei/koa-demo.git
```

安装依赖

```
npm install
```

进入server目录，运行下面命令启动服务器

```
node app.js
```

再运行下面命令启动前台项目

```
npm run dev
```
